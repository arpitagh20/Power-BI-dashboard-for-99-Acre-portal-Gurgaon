# Power-BI-dashboard-for-99-Acre-portal-Gurgaon
This is the data of 99 Acre Portal for Gurgaon location. Data contains the property listed in 2 months by channel partners. Understanding the market , by analyzing the location, price per sqft, configuration, superbuildup area  and brands.
SHORT DESCRIPTION
This dashboard provides a comprehensive overview of real estate portal activity on the 99Acres platform. It tracks key metrics such as property listings, registration dates, user activity, and property performance across different locations. The dashboard enables stakeholders to monitor trends, analyze listing behavior, and evaluate platform engagement through interactive filters and visualizations. It helps management make data-driven decisions related to property marketing, user engagement, and portal performance.

TECHNOLOGIES
